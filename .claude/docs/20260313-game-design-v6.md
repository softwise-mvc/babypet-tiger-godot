## 一句話定位（v6 更新）

> **「養你的毛小孩，有虎爺守護，每天都有牠在等你回來」**
> 以情感召喚為黏著核心，推播通知是「寵物在等你」的情感連結，不是系統通知

---

## 設計核心痛點解決

| 痛點 | v5 問題 | v6 解法 |
|------|---------|---------|
| 機械化重複 | 每天做一樣的動作 | 每日特殊事件（虎爺召喚）打破節奏 |
| 情感連結薄弱 | 照顧動作無回饋感 | 對話框 + 動畫 + 推播用寵物第一人稱說話 |
| 照顧感不足 | 開 App 才知道狀態 | 主動推播讓玩家「被需要」 |

## 二、資源系統（雙貨幣線）

| 資源 | 圖示 | 取得方式 | 用途 |
|------|------|---------|------|
| **URPoint** | 🪙 | 完成日常照顧動作 | 購買飼料/水/玩具、解鎖互動小遊戲 |
| **香火幣** | 🕯️ | 呼叫虎爺解決特殊事件 | 升級虎爺神壇、廟宇旅行費用、強化道具 |

**設計原則：**
- URPoint = 日常消耗品，容易取得，頻繁流通
- 香火幣 = 稀有資源，不互通，代表虎爺庇佑的珍貴感
- 兩條線分開管理，不可互換

## 三、核心日常循環

```
玩家開 App（被推播召喚或主動回來）
    │
    ├─ 1. 查看寵物狀態（飢餓 / 口渴 / 心情 / 清潔）
    │       對話框：寵物用第一人稱說目前狀態
    │
    ├─ 2. 主動照顧（玩家發起）
    │       餵食 → +URPoint ×2
    │       補水 → +URPoint ×1
    │       玩耍小遊戲 → +URPoint ×3 + 屬性訓練
    │       清潔 → +URPoint ×1
    │       每個動作完成 → 動畫 + 寵物對話框反應
    │
    ├─ 3. 每日特殊事件（每天 1 次隨機觸發）
    │       對話框出現事件描述
    │       玩家點擊「呼叫虎爺」
    │       虎爺出場動畫 → 解決事件
    │       +香火幣 ×3（普通）/ ×5（稀有）
    │
    ├─ 4. 互動小遊戲（選擇一種，訓練屬性）
    │       每天可玩 3 次（免費）
    │
    └─ 5. 香火幣用途（可選）
            升級神壇 / 送寵物出發廟宇旅行
```

## 四、推播通知系統（情感黏著核心）

**設計原則：推播是「寵物在等你」，不是系統通知**

| 觸發條件 | 推播文字（寵物/虎爺口吻）|
|---------|--------------------------|
| 飼料不足 | 「主人⋯我肚子好餓，你在哪裡？🥺」 |
| 飲水不足 | 「虎爺說主人今天還沒回來，我在等你⋯」 |
| 心情低落 | 「〔寵物名〕今天好像不太開心，需要你陪牠」 |
| 特殊事件待解決 | 「虎爺感應到異常！需要主人馬上回來處理！」 |
| 廟宇旅行回來了 | 「〔寵物名〕從三峽祖師廟回來了，帶了東西給你！」 |
| 每日打卡提醒 | 「虎爺每天都在守護，主人今天還沒來看〔寵物名〕呢」 |

**推播頻率規範：**
- 最多每 4 小時一次（避免騷擾）
- 飢餓/口渴各觸發一次後，同類型當日不重複
- 廟宇旅行回來立即推播（玩家主動發起的等待）

## 五、每日特殊事件系統（虎爺召喚）

### 呈現方式：全程對話框 + 小動畫，不需要戰鬥系統

```
[第一幕：事件對話框]
  插圖：房間角落出現陰影 / 寵物躲起來 / 奇怪光影
  文字：「深夜，〔寵物名〕突然豎起毛髮，低聲嗚叫...
         角落傳來奇怪的聲音...」

[選項] 玩家點擊「呼叫虎爺」按鈕

[第二幕：虎爺出場]
  動畫：虎爺從神壇跳出，金光閃爍，腳踏蓮花
  虎爺台詞：「爾等宵小，休想在此作怪！」

[第三幕：解決動畫]
  虎爺咬住邪靈 / 踩碎穢氣 / 掃除霉運
  特效：金光爆發

[結算畫面]
  「虎爺已護送邪靈離去，今晚平安！」
  +香火幣 ×3
  寵物心情恢復，跑來蹭玩家
```

### 事件類型池

| 事件 | 頻率 | 香火幣獎勵 | 說明 |
|------|------|-----------|------|
| 邪靈入侵 | 常見 60% | +3 | 基本事件 |
| 好兄弟來訪 | 中等 25% | +4 | 虎爺禮貌送走，有趣對話 |
| 財神路過 | 稀有 10% | +5 + URPoint ×10 | 大獎事件 |
| 節慶限定活動 | 節慶期間 | +8 | 中元/春節/媽祖生日 |
| 壞天氣照顧 | 中等 5% | +3 | 颱風天、過年等情境 |

## 六、互動小遊戲 × 屬性訓練系統

### 屬性影響廟宇旅行成果

| 小遊戲 | 操作方式 | 訓練屬性 | 廟宇旅行加成 |
|--------|---------|---------|------------|
| 逗貓棒 / 丟球 | 點擊節奏 / 簡單滑動 | **敏捷** | 旅行速度縮短 |
| 餵零食 / 按摩 | 點擊指定部位 | **親密** | 帶回道具品質提升 |
| 虎爺祈福小儀式 | 照順序點擊 | **靈感** | 稀有籤詩機率提升 |
| 定期清潔 / 梳毛 | 滑動清潔 | **福氣** | 旅行帶回香火幣加成 |

**規範：**
- 每天每種小遊戲各可玩 1 次免費（共 4 次）
- 小遊戲完成後：+URPoint + 對應屬性 +1
- 屬性上限：Level 10（有視覺化 icon 顯示等級）

## 七、廟宇旅行系統（被動等待 × 驚喜揭曉）

### 旅行流程

```
[出發準備]
玩家消耗香火幣 × 數量（依距離不同）
在台灣地圖選擇目的地廟宇

[等待期間]（被動，不需開 App）
1～8 小時（依寵物敏捷屬性縮短）
偶爾推播：「〔寵物名〕在廟宇遇到了一件有趣的事...」

[回來揭曉]
推播通知：「〔寵物名〕回來了！」
玩家開 App → 對話框：旅行故事（2-3 句）
翻牌 / 開箱動畫揭曉帶回收穫
```

### 廟宇列表與特色

| 廟宇 | 地點 | 旅行時間 | 特色收穫 |
|------|------|---------|---------|
| 龍山寺 | 台北 | 2 小時 | 籤詩、平安符 |
| 行天宮 | 台北 | 2 小時 | 香火幣加成 |
| 三峽祖師廟 | 新北 | 3 小時 | 稀有裝扮碎片 |
| 鹿港天后宮 | 彰化 | 4 小時 | 媽祖系道具 |
| 南鯤鯓代天府 | 台南 | 5 小時 | 稀有圖鑑 |
| 東港東隆宮 | 屏東 | 6 小時 | 王爺系限定道具 |

### 旅行收穫類型

| 收穫 | 機率 | 說明 |
|------|------|------|
| 香火幣 | 60% | 基本回報 |
| 廟宇圖鑑碎片 | 50% | 集滿解鎖廟宇介紹 |
| 虎爺裝扮碎片 | 20% | 稀有外觀收集 |
| 籤詩 | 30% | 進入收藏，有台灣文化解說 |
| 稀有道具 | 10% | 寵物房間裝飾 |

## 八、圖鑑收集系統（長期目標）

| 圖鑑類別 | 解鎖方式 | 數量目標 |
|---------|---------|---------|
| 🏯 台灣廟宇圖鑑 | 送寵物去該廟宇旅行 | 20+ 座廟宇 |
| 🐯 虎爺裝扮圖鑑 | 特殊事件 + 香火幣兌換 | 15+ 套裝扮 |
| 📜 籤詩收藏 | 廟宇旅行帶回 | 60 支籤詩 |
| 🎉 節慶道具圖鑑 | 限定活動期間取得 | 依節慶 |
| 🌟 寵物個性圖鑑 | 照顧出不同個性組合 | 12 種個性 |

**圖鑑附加價值：**
- 每個廟宇圖鑑有真實台灣文化介紹（2-3 行）
- 籤詩有現代白話解說
- 完整度達成有額外香火幣獎勵

## 九、寵物個性系統（照顧行為 × 生肖底色）

### 個性決定方式

```
開局：玩家輸入農曆生日 或 選擇生肖
→ 決定寵物「命格」（基礎個性傾向）
→ 日常照顧行為累積「個性點數」
→ 里程碑（7天/30天/100天）觸發「個性成形事件」
```

### 個性點數系統

| 照顧行為 | 影響個性方向 |
|---------|------------|
| 常玩耍、丟球 | 活潑值 + |
| 常按摩、輕聲說話 | 撒嬌值 + |
| 不常理會、獨立生活 | 傲嬌值 + |
| 常帶去廟宇旅行 | 靈性值 + |

### 個性呈現方式
- 對話框文字風格改變
- 寵物日常動作有個性差異
- 特殊事件中寵物反應不同
- **不需要複雜視覺系統，全靠對話框文字體現**

## 十、技術實作原則（Godot 4.6）

### 呈現方式對照

| 遊戲效果 | 實作方式 |
|---------|---------|
| 寵物情感表達 | SpriteFrames 動畫 + 對話框文字 |
| 虎爺出場 | Sprite 動畫 + 金光粒子（GPUParticles2D）|
| 事件戲劇感 | 對話框序列推進（Visual Novel 風格）|
| 旅行揭曉 | 翻牌/開箱 Tween 動畫 |
| 屬性成長 | 數值 + 圖示 Level，不需複雜視覺化 |
| 推播通知 | Godot 平台通知 plugin |

### 對話框系統（核心元件）
- 全遊戲統一對話框元件：`dialogue_box.tscn`
- 支援：角色立繪 + 文字逐字顯示 + 選項按鈕
- 所有事件、旅行揭曉、照顧反應皆走此系統
- 優先實作，是整個情感系統的基礎

## 十二、與 v5 差異對照

| 項目 | v5 | v6 |
|------|----|----|
| 核心情感機制 | 被動等待 | 主動推播召喚（情感黏著）|
| 資源系統 | 單一香火幣 | URPoint（日常）+ 香火幣（稀有）雙軌 |
| 虎爺角色 | 加成/守護 | 主動召喚解決事件，有戲劇性出場 |
| 寵物個性 | 無 | 照顧行為 × 生肖底色共同塑造 |
| 技術門檻 | RPG/ACT 不確定 | 明確限定對話框 + 動畫，無複雜機制 |
